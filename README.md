# BCD 4204 — Estructuras de Datos con Python

Repositorio oficial del curso **Estructuras de Datos** impartido en Lead University.

Este curso introduce al estudiante en el análisis de algoritmos, estructuras de datos clásicas y técnicas de resolución de problemas utilizando Python. A lo largo del curso se estudiarán estructuras como listas enlazadas, pilas, colas, árboles y grafos, además de algoritmos de búsqueda, ordenamiento, recursividad y programación dinámica. 

---

# 📚 Objetivos del curso

Al finalizar el curso, el estudiante será capaz de:

- Seleccionar estructuras de datos adecuadas para resolver problemas.
- Implementar algoritmos utilizando Python.
- Analizar eficiencia temporal y espacial.
- Comparar soluciones según rendimiento y escalabilidad.
- Modelar problemas reales utilizando estructuras de datos y algoritmos.
